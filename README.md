# egypt-unlocked-legal

Public, **legal-only** static site for **Egypt Unlocked / اكتشف مصر** (Travel / Tourism).
It serves the app's Privacy Policy and Terms of Use so they have stable public
URLs for the App Store / TestFlight.

## Pages
- `index.html` — legal home (links to both)
- `privacy.html` — Privacy Policy
- `terms.html` — Terms of Use

## What this is NOT
- ❌ No app source code (the app is the private repo `egypt-unlocked-private` → `egypt-unlocked-native`).
- ❌ No Supabase config, API keys, `.env`, or secrets.
- ❌ Not the old web app (`egypt-unlocked-app`) — that was removed.
- ❌ No tracking, no analytics, no external dependencies, no scripts. Static HTML only.

## Hosting (GitHub Pages)
Create a **public** repo named `egypt-unlocked-legal`, push these files to `main`,
then enable **Settings → Pages → Source: main / root**. Public URLs:
- `https://maznsyam.github.io/egypt-unlocked-legal/privacy.html`
- `https://maznsyam.github.io/egypt-unlocked-legal/terms.html`
